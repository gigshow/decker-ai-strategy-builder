# 시계열 라벨링 알고리즘

---

## 역할

시계열 데이터를 **오브젝트(대상)**로 평가하여 라벨을 부착합니다. 시그널 발생 조건을 정의합니다.

---

## 라벨 구성

| 라벨 | 의미 |
|------|------|
| **1/2 대상 + 1/2 프라임** | 평가 대상과 평가 기준(프라임)의 조합 — 대상에 대한 평가 |
| **S** | 시그널 (Signal) |
| **T** | 터치 (Touch) — 향후 시그널 진입점. 중첩의 의미 포함 |
| **1** | 기타 상태 |

---

## 오브젝트와 스윙

- **오브젝트**: 시점에 따라 멀티스윙으로 구성됨

| 스윙 | 의미 |
|------|------|
| **A** | 메인스윙 — 트렌드의 키 방향. 평가 대상 2개 이상이 현재가 기준 위/아래에 위치할 때, 가격 상승/하락으로 평가 |
| **B** | 서브스윙 — 메인스윙의 반대 방향 조정 |
| **C** | 연결스윙 — 시장 청산·진입·리버스가 일어나는 구간 |

---

## 스윙 계산

- ab 스윙, ac 스윙, bb 스윙, bc 스윙 등 조합으로 계산
- 각 스윙의 **평가봉(2프라임)**을 현재가가 브레이크할 때 **시그널 발생**

---

## 파이프라인 내 위치

```
시계열 데이터 → [라벨링 알고리즘] → 라벨된 시그널 → State Engine (progress_pct) → 오퍼레이션 룰북
```
