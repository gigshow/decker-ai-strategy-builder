## DeckerClaw 포지셔닝

> **DeckerClaw를 입양해서 자신의 에이전트에서 사용할 수 있습니다.**

- Decker와 가재(에이전트) **협업** 이미지
- 입양 = API 연동, Telegram 사용, 샘플 포크 — 자신의 트레이딩 에이전트에 DeckerClaw를 붙여 쓸 수 있음
- OpenClaw 오픈소스 기반. 에이전트·전달력·협업 강조
