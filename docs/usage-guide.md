# 사용법 가이드

**한 페이지 요약.** 신규 사용자가 안내 없이 첫 시그널 조회 가능. *말만 하면* 시그널·포지션·주문. 한글·영문 모두 지원.

---

## 채팅 vs 대시보드

| 구분 | 채팅 (Telegram 권장, OpenClaw 스킬) | 대시보드 (decker-ai.com) |
|------|------------------------|--------------------------|
| **시그널·시세** | ✅ | ✅ |
| **상의·전략** | ✅ (이 시그널 어떻게 할까?) | ✅ |
| **포지션·주문·청산** | ✅ | ✅ |
| **자동주문·뉴스** | ✅ | ✅ |
| **폴리마켓·HL** | ✅ | ✅ |
| **연동 코드 발급** | ❌ | ✅ (decker-link-telegram) |
| **API 키·실행 모드** | ❌ | ✅ |
| **전략 빌더·포트폴리오** | ❌ | ✅ |

→ **연동은 반드시 대시보드에서.** 채팅으로 "말만 하면" 시그널·주문 가능.

## Telegram 대화 키워드 (이렇게 말해보세요)

연동 후 @deckerclawbot에 **말만 하면** 돼요. 한글·영문 모두 지원합니다. 아래 키워드로 대화를 시작해 보세요.

| 목적 | 대화 키워드 (예시) |
|------|-------------------|
| **시그널** | 비트코인 시그널 알려줘, show signal, BTC signal, ETH 시그널 |
| **시그널 전략·상의** | 이 시그널 어떻게 할까, BTC 전략 알려줘, what to do with this signal |
| **시세** | 비트코인 얼마, ETH 시세, BTC price, price |
| **포지션·수익** | 포지션 보여줘, position, portfolio, 수익현황 |
| **주문** | ETH 0.01 매수해줘, buy 0.01 ETH, BTC 매도해줘, sell BTC |
| **청산** | ETH 청산해줘, close ETH, 비트코인 청산해줘 |
| **자동주문** | 이더리움 자동주문 해줘, 자동주문 설정 보여줘, 이더리움 자동주문 끄줘 |
| **시장·비교** | 시장 상태 어때, market status, BTC ETH SOL 비교 |
| **뉴스** | 뉴스 보여줘, news, 소식 보여줘, 다이제스트 |
| **도움말** | 하이, 뭐 할 수 있어, hello, 어떻게 써, 처음인데 |

**시그널·전략 응답**: rationale(이유), strategy(전략), choices(선택지) 포함. 예: "진행도 66%. [전 TF 정렬 추세] 30% 부분 익절 제안. 목표까지 홀드 / 30% 익절 / 전량 청산"

## 3분 온보딩

1. **가입** — decker-ai.com 회원가입
2. **연동** — decker-link-telegram에서 코드 발급 → Telegram @deckerclawbot에서 `/start {코드}`
3. **바로 사용** — "비트코인 시그널 알려줘"

---

## 5단계 체험 가이드 (처음인데 / 단계별로 알려줘)

| 단계 | 체험 | 사용자 액션 | 성공 기준 |
|------|------|-------------|-----------|
| **1** | 모의거래 | 가입→연동→"ETH 0.01 매수해줘"→승인 | 주문 실행, 포지션 생성 |
| **2** | 자동주문·청산 | "이더리움 자동주문 해줘"→"ETH 청산해줘" | 규칙 설정, 포지션 청산 |
| **3** | 실거래 | 설정에서 Binance API 키 등록→"ETH 매수해줘" | Binance 주문 |
| **4** | Hyperliquid | 설정에서 HL 키, exchange_preference=HL→"HL BTC 매수해줘" | HL 주문 |
| **5** | Polymarket | 설정에서 PM 연동→"Polymarket 시장 yes 10 매수" | 예측시장 주문 |

---

## HL·Polymarket 실주문 (설정 필요)

| 거래소 | 설정 경로 | 예시 명령 |
|--------|-----------|-----------|
| **Hyperliquid** | decker-ai.com → 설정 → 거래소 API → Hyperliquid | HL BTC 0.01 매수해줘 |
| **Polymarket** | decker-ai.com → 설정 → 거래소 API → Polymarket | Polymarket 시장 yes 10 매수 |

**DEX 에이전트 전용 지갑 (HL/PM 필수)**: 메인 지갑 개인키는 절대 입력하지 말 것. 새 지갑 생성 → 거래용 자금만 입금.
