**DeckerClaw** — Decker 자체 에이전트 모델 및 OpenClaw 생태계 협업. 입양해서 자신의 에이전트에서 사용할 수 있습니다.

## 공식 마스코트 v1 (2026-05-16~) — Owl with Claw

| 파일 | 용도 |
|------|------|
| `decker_claw_owl_v1.svg` | Telegram, 웹, API, 에이전트 프로필 (현재 메인) |

**컨셉**: **올빼미 (Owl) with Claw** — Geometric flat 스타일.
- 지혜 (분석) + 야행성 (24/7 시장) + 시야 (MTF) + Claw (정확한 결정)
- 색: `--accent #00d97e` eye/claw + `--bg2 #0b0f14` feather
- SVG inline = theme token 즉시 응답

**DeckerClaw 입양** = (A) Telegram·웹 사용 (B) OpenClaw 스킬로 API 연동 (C) API 직접.
